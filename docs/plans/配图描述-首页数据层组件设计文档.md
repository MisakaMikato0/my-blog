# 配图描述 — 首页数据层组件设计文档

## 配图 1

**文档位置**：组件关系

**内容描述**：展示首页数据层组件关系。顶层为 `index.astro`，其下包含 `HomeHero`、`HomeDataLayer`、`HomeTicker`、`HomePending`。`HomeDataLayer` 内部包含访问量 `DataMetricCard`、文章信息 `DataMetricCard`、社交联系 `LogoLoop.svelte`、技能展示 `LogoLoop.svelte`，并从配置文件、内容统计工具和 `/api/count` 获取数据。

## 配图 2

**文档位置**：布局设计

**内容描述**：展示 9x9 数据层矩阵。整体为 3 行等高结构，每行高度为 3 个单位。第一层左侧为 3x3 访问量卡片，右侧为 6x3 文章信息卡片。第二层为 9x3 社交联系区域，左侧 3x3 为介绍区，右侧 6x3 为跑马灯区。第三层为 9x3 技能展示区域，左侧 3x3 为介绍区，右侧 6x3 为跑马灯区。
