# 首页明信片徽章 Rainbow Button 设计

## 目标

将首页 `home-hero__name-badge` 的虹彩效果调整为 Magic UI Rainbow Button 的完整视觉版本，同时保留徽章现有文字、尺寸、位置和对话触发逻辑。

## 实现

1. 使用参考实现的五组 HSL 色值，并按红、黄绿、蓝、青、紫的顺序组成虹彩渐变。
2. 使用三层背景：实体表面、向下透明的表面层、虹彩层；所有背景从 `border-box` 定位，并分别裁切到 `padding-box` 与 `border-box`。
3. 深色首页使用纯白表面和黑字；浅色主题使用 `#121213` 表面和白字。
4. 底部光晕使用参考实现的 `60%` 宽、`20%` 高、`-20%` 偏移与 `0.75rem` 模糊。
5. 动画保持参考实现的 2 秒线性 `background-position: 0% -> 200%` 循环。
6. 移除原有黑色投影，仅保留虹彩光晕；`prefers-reduced-motion` 下停止动画。

## 验证

执行 Astro 检查与生产构建，并在桌面和 375px 移动端检查表面颜色、边框裁切、底部光晕、文字可读性与布局稳定性。
